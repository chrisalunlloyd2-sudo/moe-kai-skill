# Moe Control — Kai 9000 skill

Installable Kai skill that lets Kai drive the local **Moe** autonomous engine
(Viper system) from chat, with an interactive control panel, "while you were
away" briefings, and persistent awareness of the user.

Install in Kai: Settings → Skills → Install from GitHub →
`chrisalunlloyd2-sudo/moe-kai-skill`
